  • PathFollow/PathFollower.js

    PathFollower.js contains implementation of PathFollower class.
    Usage: Add as an EC_Script to an entity to make it follow a path or include it from another script.
    Requirements:
    • Entity should have a realXtend.EntityController.EntityController or realXtend.EntityController.PlaceableController implementation.
    Actions:
    • pauseFollow() Pauses following.
    • resumeFollow() Resumes following.
    • followPath() Starts following.
    Dynamic attributes:
    • Path {int}
    • Direction {int}
