The SVN contains VS2008 build projects for some of the external dependencies. 
This project has the following dependencies:

- Boost 1.36.0 or newer, http://www.boost.org/ . Download a prebuilt Boost package or build it by yourself.
   x Includes and libs: SVN external. In VS2k8, set the global VS environment settings to refer to Boost 
                        includes and libraries.
   x Windows DLLs: Copy the necessary Boost DLLs to the SVN bin/ directory.

- poco-1.3.3p1. The PoCo libraries (http://pocoproject.org/) are used as dynamic libraries. T
   x Includes and libs: SVN external. Download PoCo and set the global VS environment settings to refer to 
                        the PoCo includes and libraries.
   x Windows DLLs: Prebuilt poco-1.3.3p1 versions are included in bin/.

- XMLRPC-EPI 2008-04-16, http://xmlrpc-epi.sourceforge.net/main.php?t=summary is used as a dynamic library.
   XMLRPC-EPI depends on LibIconv and eXpat. To build this, first build the expat project.
   x Includes and libs: Included in the directory xmlrpc-epi. This project is set to refer to this
                        directory for the dependencies.
   x Windows DLLs: The file xmlrpcepi.dll is included in bin/.
   
- LibIconv 1.9.2 for Windows: http://gnuwin32.sourceforge.net/packages/libiconv.htm (also see 
   http://gnuwin32.sourceforge.net/packages.html) LibIconv is located in the GnuWin32\ directory, 
   and is used as a dynamic library.
   x Includes and libs: Includes are in SVN in the directory GnuWin32\include, and libs are at GnuWin32\lib,
                        This project is set to refer to this directory for the dependencies.
   x Windows DLLs: The file libiconv2.dll is included in bin/.

- eXpat 2.0.1: http://expat.sourceforge.net/ . This is used as a dynamic library, and is located in the
   directory Expat 2.0.1/ in SVN. 
   x Includes and libs: Located in Expat 2.0.1/lib/. This project is set to refer to this directory for 
     the dependencies.
   x Windows DLLs: The file libexpat.dll is included in bin/.
   
- MD5 hashing. This project is used by building the project source files as part of the main project build.
   Contains the following four files: md5.h, md5.c, md5wrapper.h, md5wrapper.c that are located in
   OpenSimProtocol/.
